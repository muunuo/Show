# Denne nettsiden vil gi brukere muligheten til å gjøre følgende ting:
1. Legge inn hva show de
- ønsker å se
- har sett
- holder på å se
2. Få og gi anbefalinger til venner/andre brukere på show de burde se
3. Lar brukere legge inn bilder, enten for profilbilde eller for plakat til show.

Ettervært vil følgende ting også være muligt: 
4. Ransjere show man har sett på, og hvor viktig det er at venner ser på det
5. Kommentere på show og venner sine anbefalinger.
6. Kontoene på nettsiden beskyttes med krypterte passord, og brukernavn er unike til hver bruker.
7. Brukere har kontoer med bio, venneliste osv.
8. Show har sjangere og nøkkelord assosiert med de. 
9. (Usansyneligt) Show kan anbefales til brukere av nettsiden basert på hva brukeren tidligere har sett på.
